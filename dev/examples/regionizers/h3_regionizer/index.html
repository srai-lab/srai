    Regionize using H3 without buffering¶

    Polygons are not fully covered in H3 cells

    In [3]:
    Copied! 
    regionizer = H3Regionizer(resolution, buffer=False)
gdf_h3 = regionizer.transform(gdf)

ax = gdf.plot()
gdf_h3.plot(ax=ax, color="red", alpha=0.5)
plt.show()
